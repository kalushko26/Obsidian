На этом этапе нужно заметить, что существует сокращение для свойств _flex-direction_ и _flex-wrap_ — _flex-flow_. Например, вы можете заменить:

```css
flex-direction: row;
flex-wrap: wrap;
flex-flow: row wrap;
```
