```js
// Что выведет?
let number = 0;
console.log(number++);
console.log(++number);
console.log(number);
```

**Ответ

```js
// Что выведет?
let number = 0;
console.log(number++); // 0
console.log(++number); // 2
console.log(number); // 2
```
